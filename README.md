# ET Smart Investor AI

AI-powered stock intelligence platform for smarter investment decisions.

## 🚀 Features
- AI-powered stock analysis
- Real-time price charts
- Smart alert system
- Multi-agent architecture

## 🏗️ Architecture
<img width="1536" height="1024" alt="etgenai-architecture" src="https://github.com/user-attachments/assets/c0cd3609-dc8f-45a4-bc8e-54d982d67e58" />

- Frontend: React + Tailwind
- Backend: Spring Boot
- APIs: Yahoo Finance / Twelve Data
- AI: LLM (Generative AI)

## 📂 Project Structure
- frontend/ → React app
- backend/ → Spring Boot APIs

## ⚙️ Setup Instructions

### 1. Clone repo
git clone <repo-link>

### 2. Run Backend
cd backend
mvn spring-boot:run

### 3. Run Frontend
cd frontend
npm install
npm run dev
